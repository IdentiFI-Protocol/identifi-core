# Security Policy

## Supported Versions
Only the latest version of IdentiFI Protocol is currently supported for security updates.

## Reporting a Vulnerability

IdentiFI Protocol prioritizes the security of our sovereign users. If you discover a security vulnerability, please do NOT open a public issue. 

Instead, please report it via:
- Direct Message on X: @IdentifiShield
- Official Channel: identifishield@proton.me

We appreciate your help in keeping the protocol stealth and secure. Every valid report will be recognized in our future Hall of Fame.
